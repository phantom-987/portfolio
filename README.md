# 🚀Personal Portfolio : https://portfolio-seven-ivory-84.vercel.app/

![Portfolio Preview](assets/images/preview.png)

A cinematic, interactive personal portfolio website built with vanilla HTML, CSS, and JavaScript — featuring a live 3D space scene, parallax starfield, animated Earth with orbiting planets and satellites, light/dark mode, and a fully functional contact form.

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Structure |
| CSS3 + CSS Variables | Styling, theming, animations |
| Vanilla JavaScript | Logic, interactivity |
| Three.js (r128) | 3D scene — Earth, planets, starfield |
| GSAP + ScrollTrigger | Scroll animations, entrance effects |
| EmailJS | Contact form email delivery |
| GitHub Pages | Hosting |
